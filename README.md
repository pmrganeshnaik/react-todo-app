# React To-Do App

A simple to-do application built with React, TypeScript, and Tailwind CSS.

## Installation

1.  Clone the repository.
2.  Run `npm install` to install the dependencies.
3.  Run `npm run dev` to start the development server.

## Usage

Open your browser and navigate to `http://localhost:5173` (or the port shown in your terminal).

*   Add a new to-do item by typing in the input field and pressing Enter or clicking the Add button.
*   Mark a to-do item as complete by clicking the checkbox.
*   Delete a to-do item by clicking the delete button.
